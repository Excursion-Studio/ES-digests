# 预设性能控制

预设性能控制（Prescribed Performance Control, PPC）是一种**非线性控制方法**，其核心思想是在控制器设计阶段就**显式地规定**系统输出误差的瞬态和稳态性能边界，并通过数学变换将这些约束转化为无约束的控制问题。

## 核心思想

传统控制方法（如 PID）通常只关注稳态误差，而预设性能控制则要求：

1. **误差始终保持在预设边界内** —— 全时段约束
2. **收敛速度可预先设定** —— 瞬态性能可控
3. **超调量有明确上限** —— 避免过大振荡

# 性能函数（Performance Function）

性能函数是预设性能控制的核心工具，它是一个**正定的、单调递减的时变函数**，用于定义误差的最大允许边界。

## 典型形式

最常见的性能函数采用**指数衰减形式**：

$$\rho(t) = (\rho_0 - \rho_\infty) e^{-lt} + \rho_\infty$$

其中：
- $\rho_0 > 0$：初始边界，需满足 $\rho_0 > |e(0)|$
- $\rho_\infty > 0$：稳态边界，代表允许的最大稳态误差
- $l > 0$：衰减率，决定收敛速度

## 物理意义

| 参数 | 作用 |
|------|------|
| $\rho_0$ | 初始阶段允许的误差范围 |
| $\rho_\infty$ | 最终稳态精度要求 |
| $l$ | 从初始到稳态的过渡速度 |

# 误差变换（Error Transformation）

为了将受约束的误差转化为无约束变量，引入**误差变换**。

## 基本思想

定义归一化误差：

$$\varepsilon(t) = \frac{e(t)}{\rho(t)}$$

要求 $|\varepsilon(t)| < 1$，即误差始终被约束在性能边界内。

## 数学变换

引入变换函数 $T(\cdot)$，将有界变量 $\varepsilon$ 映射到无界变量 $\xi$：

$$\xi = T(\varepsilon) = \frac{1}{2} \ln\left(\frac{1 + \varepsilon}{1 - \varepsilon}\right)$$

这就是著名的 **logarithmic 变换**（对数变换）。

## 变换的逆过程

$$\varepsilon = T^{-1}(\xi) = \frac{e^{\xi} - e^{-\xi}}{e^{\xi} + e^{-\xi}} = \tanh(\xi)$$

# 控制器设计流程

预设性能控制的标准设计步骤：

**步骤 1：定义性能函数**
- 选择合适的性能函数 $
ho(t)$
- 确定初始边界 $
ho_0$、稳态边界 $
ho_\infty$ 和衰减率 $l$

**步骤 2：误差变换**
- 计算归一化误差：$\varepsilon(t) = \frac{e(t)}{\rho(t)}$
- 应用对数变换：$\xi = T(\varepsilon) = \frac{1}{2} \ln\left(\frac{1 + \varepsilon}{1 - \varepsilon}\right)$

**步骤 3：设计控制律**
- 基于变换后的无约束变量 $\xi$ 设计控制律
- 确保 $\xi \to 0$ 的收敛性

**步骤 4：稳定性分析**
- 使用 Lyapunov 方法证明系统稳定性
- 验证误差 $e(t)$ 始终满足性能约束

## 关键性质

- 当 $\xi \to 0$ 时，$\varepsilon \to 0$，进而 $e(t) \to 0$
- 由于 $\xi$ 是无约束的，可以使用标准的 Lyapunov 方法设计控制律
- 性能函数 $\rho(t)$ 保证了误差的瞬态和稳态边界

# 与其他控制方法的对比

| 特性 | PID 控制 | 滑模控制 | 预设性能控制 |
|------|---------|---------|-------------|
| 稳态误差 | 渐进收敛 | 有限时间收敛 | **预设边界内** |
| 瞬态性能 | 不可控 | 部分可控 | **完全可控** |
| 超调量 | 不确定 | 可能有抖振 | **有明确上限** |
| 收敛时间 | 不确定 | 有限时间 | **可预先设定** |
